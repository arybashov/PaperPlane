# PaperPlane

Интерактивная аэродинамическая симуляция бумажного самолётика прямо в браузере.

## Что это?

Физический движок, моделирующий полёт бумажного самолётика с учётом:
- Подъёмной силы ($C_L$)
- Аэродинамического сопротивления ($C_D$)
- Гравитации
- **Ветра и турбулентности** 🆕
- Скоростного набора/планирования

Интегрирование уравнений движения методом **Рунге-Кутты 4-го порядка (RK4)**.

## Новые фичи 🚀

### 1. Ветер и турбулентность
Задаёте скорость ветра (положительно = ветер в спину, отрицательно = встречный).  
Турбулентность добавляет случайные возмущения в полёт — самолётик "болтает".

### 2. Формы самолётиков
Выберите готовую форму, и параметры $C_L$, $C_D$, масса и площадь подставятся автоматически:
- **Стрелка** — быстрая, низкое качество, высокое сопротивление
- **Планер** — дальнобойный, высокое качество, низкое сопротивление
- **Дельта** — стабильный, средние показатели
- **Своя** — полный ручной контроль

### 3. Авто-оптимизация 🎯
Нажмите кнопку **"Оптимизация"** — скрипт переберёт сотни комбинаций скорости и угла броска, чтобы найти параметры для **максимальной дальности**. Результат подставится автоматически.

## Запуск

### Способ 1: Python-сервер (рекомендуется)

```bash
python server.py
```

Откроется браузер автоматически по адресу `http://localhost:8000`.

### Способ 2: Открыть HTML напрямую

Просто откройте `index.html` двойным кликом в браузере.

## Управление

| Параметр | Описание |
|---|---|
| **Начальная скорость** | С какой силой бросили самолётик |
| **Угол броска** | Над горизонтом |
| **Высота броска** | Откуда запускаем |
| **Масса** | Тяжёлый падает быстрее, но инерция больше |
| **Площадь крыла** | Влияет на все аэродинамические силы |
| **$C_L$** | Коэфициент подъёмной силы |
| **$C_D$** | Коэфициент сопротивления |
| **Ветер** | Скорость воздушного потока (±15 м/с) |
| **Турбулентность** | Интенсивность случайных порывов (0–2) |
| **Форма** | Пресет параметров (стрелка/планер/дельта) |

### Ключевая формула

$$K = \frac{C_L}{C_D}$$ — **аэродинамическое качество**. Чем больше K, тем дальше полетит при той же высоте.

## Уравнения движения

Учитывается скорость относительно воздуха: $\vec{V}_{air} = \vec{V} - \vec{V}_{wind}$.

$$\dot{x} = v_x, \quad \dot{y} = v_y$$

$$\dot{v}_x = \frac{-D\cos\gamma - L\sin\gamma}{m} + F_{turb,x}$$

$$\dot{v}_y = \frac{-D\sin\gamma + L\cos\gamma}{m} - g + F_{turb,y}$$

где $\gamma = \arctan(v_{air,y} / v_{air,x})$ — угол наклона траектории относительно воздуха.
