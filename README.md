# PaperPlane

Интерактивная аэродинамическая симуляция бумажного самолётика прямо в браузере.

## Что это?

Физический движок, моделирующий полёт бумажного самолётика с учётом:
- Подъёмной силы ($C_L$)
- Аэродинамического сопротивления ($C_D$)
- Гравитации
- Скоростного набора/планирования

Интегрирование уравнений движения методом **Рунге-Кутты 4-го порядка (RK4)**.

## Запуск

### Способ 1: Python-сервер (рекомендуется)

```bash
python server.py
```

Откроется браузер автоматически по адресу `http://localhost:8000`.

### Способ 2: Открыть HTML напрямую

Просто откройте `index.html` двойным кликом в браузере.

## Управление

| Параметр | Описание |
|---|---|
| **Начальная скорость** | С какой силой бросили самолётик |
| **Угол броска** | Над горизонтом |
| **Высота броска** | Откуда запускаем |
| **Масса** | Тяжёлый падает быстрее, но инерция больше |
| **Площадь крыла** | Влияет на все аэродинамические силы |
| **$C_L$** | Коэффициент подъёмной силы |
| **$C_D$** | Коэффициент сопротивления |

### Ключевая формула

$$K = \frac{C_L}{C_D}$$ — **аэродинамическое качество**. Чем больше K, тем дальше полетит при той же высоте.

## Уравнения движения

$$\dot{x} = v_x, \quad \dot{y} = v_y$$

$$\dot{v}_x = \frac{-D\cos\gamma - L\sin\gamma}{m}$$

$$\dot{v}_y = \frac{-D\sin\gamma + L\cos\gamma}{m} - g$$

где $\gamma = \arctan(v_y / v_x)$ — угол наклона траектории.
